# Student Score Analyzer

A beginner-friendly Python project that analyzes student records using core Python data structures.

## Concepts Used

- Lists
- Tuples
- Dictionaries
- Sets
- Loops
- Conditionals

## Features

- Counts total student records
- Finds unique students and subjects
- Counts records per student
- Computes average score per student
- Finds the highest scoring record
- Lists top performers with scores >= 90

## Dataset Format

Each record is stored as:

(student_name, subject, score)

Example:

("Alice", "Math", 85)

## How to Run

```bash
python main.py
