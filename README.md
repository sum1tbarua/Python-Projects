## Lesson 1 вЂ“ Log Analyzer
Concepts practiced:

- Variables
- Loops
- Conditional statements
- File reading
- Basic string operations

Project:

A simple **log analyzer** that reads system logs and counts occurrences of:

- INFO
- WARNING
- ERROR

This simulates real-world **log analysis used in backend systems and ML experiment tracking**.

## Lesson 2 вЂ“ Student Score Analyzer

Concepts practiced:

- Lists
- Tuples
- Dictionaries
- Sets
- Data aggregation

Project features:

- Count total records
- Identify unique students
- Identify unique subjects
- Count records per student
- Compute average score per student
- Find the highest scoring record
- Identify top performers

This project demonstrates how Python data structures are used to analyze structured data.
